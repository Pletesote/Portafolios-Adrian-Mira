Proyectos Personales

Desarrollo Full-Stack / Lua / JS

Sistema Avanzado de Personalización de Objetos en Fivem

Desarrollo de un ecosistema compuesto por dos recursos interconectados (`nebula_personalizacion` y `nebula_imghost`) para servidores de Roleplay. Este sistema soluciona el problema de depender de enlaces externos para las imágenes dentro del juego, permitiendo una inmersión completa al simular tiendas de diseño reales.

  • Microservicio NUI (`nebula_imghost`): Implementación de una interfaz gráfica (HTML/JS) que intercepta el portapapeles (Ctrl+V). Utiliza un <canvas> invisible para redimensionar dinámicamente la imagen y codificarla a Base64 WebP, optimizando drásticamente la carga de la base de datos.
  • Backend y Lógica (`nebula_personalizacion`): Script en Lua conectado a MySQL. Gestiona permisos dinámicos por rango de trabajo, creación de zonas de interacción (Raycasting con ox_target), consumo de inventario y un panel de administración para aprobar/denegar diseños.
  • Modo Developer e Integración: El script incluye un entorno de desarrollo estructurado que separa la administración técnica del rol de los jugadores (IC), manteniendo el panel de monitoreo activo tras reinicios y enviando logs detallados del admin menu de forma invisible para los usuarios.
  • Intercomunicación: Uso avanzado de exports y NUICallbacks para asegurar la comunicación fluida y segura entre la interfaz web asíncrona y la lógica del servidor.
